XLSX files of "Medicine" data from 2005 through 2015 have been collected from [SciMago] (http://www.scimagojr.com/countryrank.php), an organization that collects aggregate data on research output by country.

The following country names have been converted to their more common names (and more frequently used on Pubmed)

"Russian Federation" -> "Russia"  
"Viet Nam" -> "Vietnam"  
"Palestine " -> "Palestine"  
"Zambia " -> "Zambia"  
"Côte d’Ivoire" -> "Ivory Coast"  
"Haïti" -> "Haiti"
